综上所述，为解决这些问题，需要有⼀个平台满⾜⽤户快速寻找车位的需求。 <br/>

需求：  

1. 考虑实际路况和距离等因素为⽤户推荐满⾜条件的停车位 。

4. ⽤户可以对停车场和停车时长根据⾃⼰的需求进⾏选择 。<br/>

1. ⽤户：有停车需求的⽤户
2. 客户：停车场管理部门
